Seja bem-vindo(a) ao AcheiUnB, {{ name }}!

Estamos muito felizes por ter você aqui! O AcheiUnB é o lugar ideal para ajudar os estudantes da UnB a encontrar e recuperar itens perdidos de forma rápida, organizada e segura.

Como funciona?

  • Login: A autenticação é feita pela Microsoft, logo, somente discentes dentro do domínio da UnB podem utilizar o site.
  • Cadastrar itens: O usuário é capaz de adicionar informações sobre itens perdidos ou encontrados preenchendo os campos destinados.
  • Buscar itens: É possível usar a nossa barra de pesquisa para encontrar o que foi perdido ou para devolver o que foi encontrado.
  • Conectar com outros: Os usuários são capazes de conversar diretamente uns com os outros pelo nosso chat integrado.
  • Ver meus itens: O estudante pode visualizar seus itens postados, tanto perdidos, quanto encontrados.
  • Denunciar: Denúncias são possíveis de serem feitas pelo site, onde o usuário é direcionado ao envio de um email para a nossa equipe.

Se precisar de ajuda, nossa equipe está sempre à disposição! Não hesite em entrar em contato.

Acesse agora o AcheiUnB e aproveite!
